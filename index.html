4th Year Project

For my 4th Year Project I am aiming to create a new application based off the Dublin GAA website.

My application will scrape data from the website such as fixtures and results for various different competitions. From this users will be able to filter matches and request data visualisations regarding performances in previous matches.

There will also be addition functionality such as a squad selector where a user can choose a starting 15 players for a match from a collection of squad players.

3rd Year Project

Throughout the second semester of 3rd Year I was working within the Department of Social Protection. It was during this time that myself and 2 other students from IT Tallaght undertook a project based on the usefullness of the Visual Studio Tool ReSharper and how it can be used within a team enviroment to enforce coding standards.

The main project was divided into 3 main sections

  • Investigation of Coding Standards
  • Investigation of ReSharper
  • Implementation of ReSharper

I was assigned the lead position for the Implementation of ReSharper. This involved downloading and installing ReSharper on my local machine so that I could customise the different settings so that the coding standards for the DSP could be enforced. From the investigation in the other 2 sections I was able to understand which coding standards should be enforced and how each should be customised so that they agreed with the actual work that was going on in DSP. This meant some settings such as naming conventions had to be altered to allow some capitals as the developers worked with a lot of abbreviations that were used on a day to day basis in the DSP.

2nd Year Project

In my second year of college I was asked to complete a project in either Java Swing or Play framework. I chose Java Swing as I felt I had stronger software development skills as opposed to web development. The idea for the project was to create a piece of software that could be used by a company to create a staff roster and keep track of staff holidays.

The project itself was enjoyable to work on as learning about Java Swing was interesting as it wasn't being tought as part of the college curriculum any more. I soon learned how it was becoming outdated and the choice of Swing over Play Framework was probably incorrect looking back.

My role in this project was the functionality of logining in as both a normal user and admin user. These users represented an employee and employer. From here I created the functionality for a user to be able to create a new employee by completing a form which was then sent to a database where it would be stored. The employer could then retrieve a list of all employees and edit or delete an employee.
